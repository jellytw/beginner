# Basic GitHub project

## A simple palindrome checker in Python(used to learn concourse CI build)

### Usage

* Clone the repository

```
git clone https://github.com/superswan/beginner.git 
```

* Change directory

```
cd beginner
```

* Execute code

```
python palindrome.py
```
